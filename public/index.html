API Routes

Users Table

path method description
/users GET Get Users
/users/:id GET Get User By Id
/users/register POST Add/Register A User
/users/login POST User Login
/users/update/:id PATCH Edit Users
/users/delete/:id DELETE Remove User

Items Table

path method description
/items GET Get Items
/items/:id GET Get Item By Id
/items/addItems POST Add a item
/items/update/:id PATCH Edit Item
/items/delete/:id DELETE Remove Item

Orders Table

path method description
/orders GET Get orders
/orders/:id GET Get Order By Id
/orders/addOrder POST Add an Order
/orders/update/:id PATCH Edit Products
/orders/delete/:id DELETE Remove Order
/orders/getOrderByUser/:id GET Get a user's order
